--- layout: portfolio year: 2015 title: DocSystem tagline: A Google Drive-like interface for categorisable documents and fine-tuned access control banner: docsystem.png images: - docsystem.png tags: - PHP - MySQL - Foundation - jQuery - LDAP --- DocSystem is an enterprise-level document management system written in PHP for a client in 2015. Features include: 1. Many-to-many document categorisation 2. AJAX search/sort/filter 3. Role-based user access control for individual files and categories, which can be inherited from parent categories 4. Version control for each document 5. LDAP integration
